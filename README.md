# Oracle PL/SQL Lab Portfolio
 
A collection of Oracle PL/SQL lab exercises completed as part of my university coursework. Each lab builds on the previous one, covering core procedural SQL programming concepts using Oracle's HR sample schema.
 
---
 
## Lab 3 — PL/SQL Basics: Variables, Control Flow & Loops
 
Introduces the fundamentals of PL/SQL programming. The exercises cover variable declaration using `%TYPE` and `%ROWTYPE` anchoring, `SELECT INTO` queries, conditional logic with `IF/ELSE` and `CASE` statements, and iterative processing with `FOR` and numeric loops. Also includes basic DML (`UPDATE`) within a PL/SQL block and the use of built-in functions like `SYSDATE`, `EXTRACT`, and `TO_CHAR`.
 
---
 
## Lab 4 — Collections: VARRAYs, Records & Exception Handling
 
Focuses on PL/SQL collection types and structured data. Covers `VARRAY` declaration and initialization, user-defined `RECORD` types, and combining records with arrays to store multi-column data. Introduces exception handling with `NO_DATA_FOUND` and `TOO_MANY_ROWS`. Also uses `DBMS_RANDOM` for generating random values and `%ROWTYPE` for full-row record storage.
 
---
 
## Lab 5 — Advanced Collections: Nested Tables, Associative Arrays & BULK COLLECT
 
Covers the remaining PL/SQL collection types and performance-oriented data loading. Introduces nested tables with `BULK COLLECT INTO` for efficient multi-row fetching, and associative arrays (`INDEX BY PLS_INTEGER`) for key-based data storage. Demonstrates practical patterns such as copying filtered data between tables, and iterating associative arrays with `FIRST` / `NEXT`.
